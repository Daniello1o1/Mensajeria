## Abstract
This is an app developed in Android Studio that implements a realtime database using Firebase to be able to send and recieve messages from users who have the app installed.
The interface is highly inspired in Whatsapp to be friendly with the user.
## Installation
1. Install Android Studio
2. Click on "Get from VCS" and select "Git"
3. Copy the link from this repository and paste it in the URL text field
4. Click in "Clone"
5. It may ask to login from your github account
6. Once is finished Android Studio will open the project automatically
7. You can press the "Run" button to test the project in an Android emulator or you can deploy direclty if you have a phone connected
